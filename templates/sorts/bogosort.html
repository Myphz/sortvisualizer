The Bogo Sort algorithm (also called Stupid Sort) is an iterative sorting algorithm particularly inefficient. It's based on randomly shufflying the elements of the data structure and then check if they are ordered. If not, repeat this process.

It is a probabilistic algorithm. It can be proven mathematically that it will arrive at the solution on average in about n x n! iterations, while the worst case performance is not measurable.
